# Food Expiration Date Tracker - Home Assistant Add-on

Run this Telegram bot as a Home Assistant add-on on HAOS.

## Install

Replace `YOUR_GITHUB_USERNAME` in the link below with your real GitHub username (or org), then open it:

[![Add repository to Home Assistant](https://my.home-assistant.io/badges/supervisor_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%iSunRise%2Ffood_exp_date_tracker)

After adding the repository:
1. Open the add-on store in Home Assistant.
2. Install **Food Expiration Date Tracker**.
3. Set required options:
   - `telegram_bot_token`
   - `database_url`
   - `openrouter_api_key`
4. Start the add-on.
